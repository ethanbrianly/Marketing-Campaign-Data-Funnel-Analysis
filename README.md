# Marketing-Campaign-Data-Funnel-Analysis

## Overview
This project analyzes marketing campaign data to evaluate funnel performance and identify trends across different audiences, regions, and advertising platforms.

The analysis tracks the customer journey from impressions → clicks → leads → applications → enrollments to measure campaign effectiveness and conversion performance.

## Tools Used
- SQL
- Power BI
- Excel

## Project Goals
- Analyze marketing funnel performance across campaigns
- Calculate key performance indicators (KPIs) such as:
  - Click-through rate (CTR)
  - Lead conversion rate
  - Application conversion rate
  - Enrollment conversion rate
- Compare campaign performance across:
  - Target audience demographics
  - Geographic regions
  - Marketing platforms

## Dashboard Features

### Funnel Analysis
- Visualized the full marketing funnel from impressions to enrollments
- Identified drop-off points between each stage of the customer journey

### Audience Analysis
- Compared average clicks, leads, applications, and enrollments across age groups
- Evaluated which audiences produced stronger conversion rates

### Regional Analysis
- Analyzed campaign performance by region
- Compared conversion rates and ROI percentages across locations

### Platform Analysis
- Evaluated performance across platforms including:
  - LinkedIn
  - Google Ads
  - Facebook
  - YouTube
  - Instagram
- Identified top-performing platforms based on engagement and conversion metrics

## Key Insights
- Analyzed millions of impressions and tracked movement through the marketing funnel
- Compared platform efficiency using CTR and conversion rates
- Highlighted differences in audience and regional performance to support data-driven decisions
